# Product

## Register

product

## Users

One traveler using a phone or desktop to plan cross-border trips with an AI travel butler. The user wants a private-trip-planner feeling: route clarity, budget control, transit evidence, and reusable prompt workflows.

## Purpose

Provide a one-stop lightweight travel intelligence site: import trip facts, render a route spine, attach map evidence, track fixed/flexible budgets, and expose compact prompts for AI-assisted planning without wasting tokens.

## Principles

- Route first: hotels are bases, not route nodes.
- Evidence beside claims: traffic screenshots and budget notes stay next to each leg.
- Reusable trip pack: data schema should work for Japan now and any cross-country trip later.
- Token thrift: each AI prompt asks for structured deltas, not full re-planning.
